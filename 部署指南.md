# 小说生成器部署指南

## 推荐部署方案（适合中国用户）

### 方案一：Zeabur（推荐）
- 支持微信/支付宝支付
- 中文界面
- 访问：https://zeabur.com

**步骤：**
1. 注册 Zeabur 账号
2. 创建新项目
3. 连接你的 GitHub 仓库
4. 添加 PostgreSQL 数据库服务
5. 配置环境变量（参考下方）
6. 部署

### 方案二：Railway（国际化平台）
- 支持信用卡支付
- 访问：https://railway.app

**步骤：**
1. 使用 GitHub 账号登录
2. New Project → Deploy from GitHub repo
3. 选择你的仓库
4. 添加 PostgreSQL 插件
5. 配置环境变量
6. 部署

### 方案三：本地 Docker 部署
如果你有自己的服务器或想在本地运行：

```bash
cd docker
cp .env.example .env
# 编辑 .env 文件，填入你的配置
docker-compose up -d
```

## 必需的环境变量

```
# 数据库
DATABASE_URL=postgresql://user:password@host:5432/dbname

# JWT密钥（随机字符串）
JWT_SECRET=your-random-secret-key

# AI配置（选择一个）
OPENAI_API_KEY=your-openai-key
ANTHROPIC_API_KEY=your-anthropic-key
DEEPSEEK_API_KEY=your-deepseek-key

# 前端地址
FRONTEND_URL=https://your-frontend-url.com
```

## 部署后设置

1. 访问你的应用地址
2. 注册账号
3. 在 AI 配置页面填入你的 API 密钥
4. 开始创作！

## 常见问题

**Q: 哪个 AI 服务最便宜？**
A: DeepSeek 对中国用户最友好，价格便宜且支持支付宝

**Q: 数据库怎么初始化？**
A: 应用首次启动会自动创建表结构

**Q: 如何更新代码？**
A: 推送到 GitHub 后，Zeabur/Railway 会自动重新部署
